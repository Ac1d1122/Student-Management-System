*Student Management System Developed By Pritam Das*



In the main menu the user will be shown 7 options:-

Press 1 for entering data
Press 2 for showing data
Press 3 for searching data
Press 4 for updating data
Press 5 for deleting data
Press 6 for exiting the program
Press 7 for exporting the database

When you chose option 7 your data in the database will be saved in the same folder as the program in a file named "database.txt".
